### Step 1: Download the Extension

The extension files are located in the `chrome-extension` folder of the IronSyde project.

### Step 2: Enable Developer Mode in Chrome

1. Open Chrome and navigate to `chrome://extensions/`
2. Toggle **Developer mode** ON (top right corner)

### Step 3: Load the Extension

1. Click **Load unpacked**
2. Select the `chrome-extension` folder
3. The IronSyde extension should now appear in your extensions list

## Usage

### Getting AI Suggestions

1. Open OnlyFans and navigate to your messages
2. When you receive a message, the IronSyde overlay will appear

## Troubleshooting

### Extension Not Working

1. Refresh the OnlyFans page
2. Check that you're logged into IronSyde
3. Verify your API key is valid
4. Check the browser console for errors

### AI Suggestions Not Appearing

1. Make sure you're on the OnlyFans messages page
2. Check that the extension is enabled
3. Try refreshing the page
4. Verify your IronSyde subscription is active

### Can't Connect

1. Verify your API key is correct
2. Check your internet connection
3. Make sure IronSyde is accessible
4. Try generating a new API key
